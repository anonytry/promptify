# Contributing to Promptify

Thank you for your interest in contributing to Promptify!

## How to Contribute

1.  **Report Bugs:** Use the GitHub Issue tracker.
2.  **Suggest Features:** Open a new issue with the "feature request" tag.
3.  **Submit Pull Requests:**
    *   Fork the repository.
    *   Create a new branch for your feature or fix.
    *   Ensure your code follows the existing style.
    *   Submit a PR with a clear description of changes.

## Engineering Standards

*   **Bash/Zsh:** Follow Shellcheck recommendations.
*   **Modularity:** Keep logic separated into modules.
*   **Portability:** Maintain compatibility with Termux and Linux.
*   **Performance:** Avoid slow subshells in the UI path.
